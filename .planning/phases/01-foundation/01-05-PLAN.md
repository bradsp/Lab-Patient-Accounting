# Phase 01-05: Modal and Utility Components

## Objective
Create modal dialog, loading indicators, and other utility components needed throughout the application.

## Context

@Lab PA WinForms UI/Library/InvoiceWaitForm.Designer.cs
@Lab PA WinForms UI/Forms/ProgressForm.Designer.cs

## Tasks

### Task 1: Create Modal Dialog Component
**Type:** create
**Files:** `LabOutreachUI/Components/Modals/Modal.razor`
**Action:**
- Create reusable modal component
- Support title, body content (RenderFragment), and footer
- Support sizes (sm, md, lg, xl)
- Support Show() and Close() methods
- Handle backdrop click to close (configurable)
- Handle escape key to close
- Use Bootstrap modal classes

**Verify:** Modal opens, displays content, closes properly

### Task 2: Create Confirmation Dialog Component
**Type:** create
**Files:** `LabOutreachUI/Components/Modals/ConfirmDialog.razor`
**Action:**
- Create confirmation dialog for delete/destructive actions
- Support customizable message

- Support warning/danger styling options

**Verify:** Confirmation shows, returns true/false based on button clicked
