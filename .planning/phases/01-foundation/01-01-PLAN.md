# Phase 01-01: Navigation and Layout Structure

## Objective
Extend the existing LabOutreachUI MainLayout and NavMenu to support the full patient accounting application with module-based navigation.

## Context
@LabOutreachUI/Shared/MainLayout.razor
@LabOutreachUI/Shared/NavMenu.razor
@Lab PA WinForms UI/Forms/MainForm.Designer.cs

## Tasks

### Task 1: Extend NavMenu with Billing Module
**Type:** create
**Files:** `LabOutreachUI/Shared/NavMenu.razor`
**Action:**
- Add "BILLING" section header to navigation
- Add menu items: Account, Work List, Charge Entry, Batch Charges, Remittance, Claims, Client Invoices, Collections
- Use appropriate icons from Open Iconic
- Wrap billing items in AuthorizeView with appropriate policy

**Verify:** Navigation renders with Billing section visible to authorized users

### Task 2: Add Dictionaries Module to NavMenu
**Type:** modify
**Files:** `LabOutreachUI/Shared/NavMenu.razor`
**Action:**
- Add "DICTIONARIES" section header
- Add menu items: Audit Reports, Charge Master, Clients (extend existing), Insurance Plans, Physicians, Pathologists
- Apply consistent styling with existing sections

**Verify:** Dictionaries section renders below Billing section

### Task 3: Add System Administration Module to NavMenu
**Type:** modify
**Files:** `LabOutreachUI/Shared/NavMenu.razor`
**Action:**
- Add "ADMINISTRATION" section header
- Add menu items: User Security, System Parameters, Interface Mapping, Interface Monitor, Log Viewer, Account Locks
- Wrap in AuthorizeView for admin-only access

**Verify:** Administration section only visible to admin users

### Task 4: Enhance MainLayout with Status Bar
**Type:** modify
**Files:** `LabOutreachUI/Shared/MainLayout.razor`
**Action:**
- Add footer status bar showing: current user, database connection name
- Style consistently with existing top row
- Use Bootstrap 5 utility classes

**Verify:** Status bar displays at bottom with user and database info

### Task 5: Create RecentAccounts Component
**Type:** create
**Files:** `LabOutreachUI/Components/Shared/RecentAccounts.razor`
**Action:**
- Create component to display recently accessed accounts
- Store recent accounts in session/local storage
- Display as dropdown or quick-access list
- Limit to 10 most recent

**Verify:** Recent accounts component stores and displays accessed accounts
