# Phase 01-02: Form Input Components

## Objective
Create reusable Blazor form input components that mirror the WinForms custom controls and provide consistent validation and formatting.

## Context
@LabOutreachUI/Shared/AutocompleteInput.razor
@Lab PA WinForms UI/UserControls/CurrencyTextBox.Designer.cs
@Lab PA WinForms UI/UserControls/DateTextBox.Designer.cs
@Lab PA WinForms UI/UserControls/LabeledTextBox.Designer.cs

## Tasks

### Task 1: Create CurrencyInput Component
**Type:** create
**Files:** `LabOutreachUI/Components/Inputs/CurrencyInput.razor`
**Action:**
- Create component with decimal binding
- Format as currency on blur ($ prefix, 2 decimal places)
- Parse currency input (allow with/without $ sign)
- Support positive-only validation option
- Include validation message display
- Use Bootstrap form styling

**Verify:** Component formats and parses currency correctly, validates positive amounts

### Task 2: Create DateInput Component
**Type:** create
**Files:** `LabOutreachUI/Components/Inputs/DateInput.razor`
**Action:**
- Create component with DateTime? binding
- Support date picker (HTML5 date input)
- Support manual entry with mm/dd/yyyy format
- Parse various date formats
- Validate date ranges (optional min/max)
- Display placeholder text

**Verify:** Date input accepts picker and manual entry, formats consistently

### Task 3: Create PhoneInput Component
**Type:** create
**Files:** `LabOutreachUI/Components/Inputs/PhoneInput.razor`
**Action:**
- Create component with string binding
- Apply phone number mask (###) ###-####
- Auto-format as user types
- Strip formatting for storage
- Validate phone format

**Verify:** Phone input formats correctly, stores unformatted value

### Task 4: Create SSNInput Component
**Type:** create
**Files:** `LabOutreachUI/Components/Inputs/SSNInput.razor`
**Action:**
- Create component for Social Security Number
- Apply mask ###-##-####
- Option to mask display (show only last 4)
- Validate SSN format

**Verify:** SSN input masks correctly, supports partial display

### Task 5: Create ZipCodeInput Component
**Type:** create
**Files:** `LabOutreachUI/Components/Inputs/ZipCodeInput.razor`
**Action:**
- Create component for ZIP codes
- Support 5-digit and ZIP+4 format
- Auto-format with hyphen for ZIP+4
- Validate ZIP format

**Verify:** ZIP input handles both formats, validates correctly

### Task 6: Create FormField Wrapper Component
**Type:** create
**Files:** `LabOutreachUI/Components/Inputs/FormField.razor`
**Action:**
- Create wrapper component for consistent field layout
- Include label, input slot, validation message
- Support required indicator
- Support help text
- Use Bootstrap form-group styling

**Verify:** FormField provides consistent layout for all input types
